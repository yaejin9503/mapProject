# 공공임대주택 지도 서비스

### 왜 만들게 되었는가?

기존에 한 페이지에서 확인이 어려웠던 공공주택 정보를
하나의 페이지에서 확인할 수 있도록 하고싶어서 만들게 되었습니다.
